Looping in Ruby and JavaScript (comparisons)

The simplest way to create a loop in Ruby is using the keyword loop. Loop takes a block, which is denoted {...} or do...end. A loop will execute any code within the block until you manually intervene with such keyworfds as the breakstatement.Inside the block is wher you can create conditional arguments like if() and else() statements. It is also where you can use enumerable properties to iterate through things such as asrrays and hashes.

The while loop is given a parameter that evaluates to a boolean. Once the boolean expression becomes false, the loop will is not executed again

$i = 0
$num = 5 while $i < $num do
puts("Inside the loop i = #$i" )
$i +=1
end

The forloop is used to loop over a collection of elements. Unlike a while loop whwere if we're not careful we can cause an infinite loop, for loops have a definite end sicne it's looping over a finite number of elements. It begins with the for reserved word, followed by a variable, then the in reserved word, and then a collection of elements.

for variable [, variable ...] in expression [do]codeend

The until loop is simply the opposite of the while loop. It runs until something is true. In other words it's anothe way to phrase the problem of a while loop.

until conditional [do]
code
end

A do/while loop is very similar to a while loop, with the only difference being that the code within the loop gets executed one time, prior to the conditional check to see if the code should be executed. In a "do/while" loop, the conditional check is placed at the end of the loop as opposed to the beginning.

Iterators are methods that naturally loop over a given set of data and allow you to operate on each element in the collection. There are many in Ruby but the most common are .each .map .inject .times

names = ['Bob', 'Joe', 'Steve', 'Janice', 'Susan', 'Helen']
names.each { |name| puts name }

In Ruby there is the break statement that enables you to terminates a method with an associated block if called within the block (with the method returning nil).

for i in 0..5
if i > 2 then
break
end
puts "Value of local variable is #{i}"
end

Loops in JavaScript are used to execute the same block of code a specified number of times or while a specified condition is true. Very often when writing code you want the same block of code to run over and over again in a row. Instead of writing the same script over and over again.

In JavaScript there are two different kinds of loops.
for loops loop through a block of code a specified number of times.
while loops loop through a block of code while a specified condition is true.

The for loop is used when you know in advance how many times the script should run.

for (var=start_value;var < value;var=var+increment) { code to be executed }

for (i=0; i < 10; i++) { document.write("The number is" + i) }

Explanation: The exampe above defines a loop that starts with zero, the loop will continue to run as long as i is less than ten, i will increase by one each time the loop runs.

Thefor each...instatement iterates a specified varibale over all values of object's properties. For each distinct property, a specified statement is exectued.

for each (variable in object) {statement}

The while loop is used when you don't know ahead of time how many time you want your loop to run. In other words, you want to loop to execute and continue executing while the specified condition is true.

while (var < end_value) do
puts("Inside the loop i = #$i" )
$i +=1
end

var i=0 while (i< 10) {do something...}

The example above defines a loop that starts with i = 0, the loop will continue to run as long as i is less then ten, i will increase by one each time.

There's also the do...while loop which is a variant of the while loop. This loop will always execute a block of code ONCE, and then it will repeat as long as the specified condition is true. This loop will always be executed at least once, even if the condition is false, because the code is executed before the condition is tested.

do{code to be executed}while (var < endvalue)

There's also the for...in statement used to iterate through the elements of an array or through the properties of an object. The code in the body of the for...in loop is executed once for each element/property.

for (variable in object)
{
code to be executed
}
